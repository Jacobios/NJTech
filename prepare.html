🎒 入学准备清单

这些东西别忘了带！

📄 证件类（最重要！）
  • ✓ 录取通知书 & 高考准考证
  • ✓ 身份证原件 + 正反面复印件×2（复印件可能后面会用）
  • ✓ 一寸/二寸证件照（红底+蓝底+白底）各10张
  • ✓ 团员证 & 团组织关系转接证明
  • ✓ 银行卡（学校后面会带着办，只办储蓄卡，别办信用卡。不着急开学提前办卡）
  • ✓ 助学贷款相关材料（如需办理询问辅导员）
💡 建议把所有证件扫描成PDF存在手机里，复印件多准备几份，总会用得到！
🛏️ 宿舍用品
  • ✓ 床上三件套×2（学校床尺寸 0.9m×2m）
  • ✓ 被子+褥子+枕头（可学校买，建议自带）
  • ✓ 蚊帐/床帘（必备！南京蚊子很凶）
  • ✓ 插线板（线长1.5m以上）
  • ✓ 台灯（建议充电款，熄灯后也能用）
  • ✓ 衣架×10-15个 + 晾衣夹
  • ✓ 洗漱用品（脸盆、毛巾、牙刷等）
  • ✓ 拖鞋（宿舍一双，洗澡一双）
⚠️ 宿舍限电！大功率电器（吹风机、电热水壶、电热毯等）属于违规电器，查到会没收+通报批评！
💻 电子产品
  • ✓ 手机 + 充电器 + 数据线×2
  • ✓ 笔记本电脑（大学天天用，建议高考考完就买）
  • ✓ 充电宝（大容量！）
  • ✓ U盘（交作业、打印必备）
  • ✓ 耳机（宿舍生活必备，尊重室友）
☀️ 军训特需
  • ✓ 防晒霜（SPF50+）
  • ✓ 大容量水杯（1L以上）
  • ✓ 鞋垫（姨妈巾鞋垫YYDS，男生别不好意思）
  • ✓ 藿香正气水/清凉油
  • ✓ 湿巾+纸巾
  • ✓ 创可贴
👔 衣物建议

南京的天气特点：夏天热、冬天冷、春秋短

  • ✓ 夏装（9月还很热，多带短袖）
  • ✓ 秋装（10月降温很快，南京是没有秋天的，本人亲身经历！）
  • ✓ 冬装（可以国庆回家带，或者让家里寄）
  • ✓ 运动鞋×2（军训费鞋）
  • ✓ 雨伞（南京梅雨季很长，写这一条信息的时候恰好就是梅雨季）
🔔 不要从学长学姐那里买被子！不要买！学校超市和网购都比那个划算且质量好！
